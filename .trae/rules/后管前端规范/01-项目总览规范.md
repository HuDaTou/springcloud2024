# 后管前端项目总览规范

> **适用范围**：`frontend/vue-vben-admin/` 目录下的所有文件

## 一、强制要求（AI必须100%遵守，违反即错误）

1. **必须使用TypeScript**：所有文件必须使用TypeScript编写，禁止使用JavaScript
2. **必须使用Vue 3 Composition API**：所有组件必须使用`<script setup>`语法
3. **必须使用Ant Design Vue (antdv-next)**：UI组件必须优先使用 `antdv-next` 组件库
4. **必须遵循目录结构**：严格按照 playground 规定的目录结构组织代码
5. **必须使用requestClient**：所有API请求必须通过 `requestClient` 或 `baseRequestClient` 发起
6. **必须使用国际化**：所有用户可见文本必须使用`$t()`函数进行国际化处理
7. **必须使用Pinia**：状态管理必须使用Pinia
8. **必须使用VXE Table**：表格组件必须使用 `useVbenVxeGrid`

## 二、推荐做法（AI优先采用，效果更好）

1. **使用组合式函数**：将可复用逻辑提取到composables中
2. **使用TypeScript类型定义**：为所有API接口、组件Props、事件定义明确的类型
3. **使用命名空间**：API接口使用namespace组织相关类型和函数
4. **使用模块化路由**：路由配置按模块拆分，使用 `import.meta.glob` 自动导入
5. **使用适配器模式**：使用adapter层封装第三方组件，统一接口
6. **使用响应式设计**：页面布局必须支持响应式，适配不同屏幕尺寸

## 三、禁止行为（AI绝对不能生成）

1. **禁止硬编码文本**：所有用户可见文本禁止直接硬编码，必须使用国际化`$t()`
2. **禁止直接使用axios**：禁止直接使用axios或fetch，必须使用requestClient
3. **禁止使用Options API**：禁止使用Vue 2的Options API语法
4. **禁止在组件中直接调用API**：API调用必须通过api层的封装函数
5. **禁止使用any类型**：禁止使用any类型，必须明确定义类型
6. **禁止在template中写复杂逻辑**：模板中的复杂逻辑必须提取到methods或computed中
7. **禁止使用全局样式污染**：组件样式必须使用scoped

## 六、标准代码示例

### 应用入口文件 (main.ts)
```typescript
import { initPreferences } from '@vben/preferences';
import { unmountGlobalLoading } from '@vben/utils';

import { overridesPreferences } from './preferences';

async function initApplication() {
  const env = import.meta.env.PROD ? 'prod' : 'dev';
  const appVersion = import.meta.env.VITE_APP_VERSION;
  const namespace = `${import.meta.env.VITE_APP_NAMESPACE}-${appVersion}-${env}`;

  await initPreferences({
    namespace,
    overrides: overridesPreferences,
  });

  const { bootstrap } = await import('./bootstrap');
  await bootstrap(namespace);

  unmountGlobalLoading();
}

void initApplication();
```

### Bootstrap (bootstrap.ts)
```typescript
import { createApp, watchEffect } from 'vue';

import { registerAccessDirective } from '@vben/access';
import { registerLoadingDirective } from '@vben/common-ui';
import { preferences } from '@vben/preferences';
import { initStores } from '@vben/stores';
import '@vben/styles';
import '@vben/styles/antdv-next';

import { useTitle } from '@vueuse/core';

import { $t, setupI18n } from '#/locales';
import { router } from '#/router';

import { initComponentAdapter } from './adapter/component';
import { initSetupVbenForm } from './adapter/form';
import App from './app.vue';

async function bootstrap(namespace: string) {
  await initComponentAdapter();
  await initSetupVbenForm();

  const app = createApp(App);

  registerLoadingDirective(app, {
    loading: 'loading',
    spinning: 'spinning',
  });

  await setupI18n(app);
  await initStores(app, { namespace });

  registerAccessDirective(app);

  app.use(router);

  watchEffect(() => {
    if (preferences.app.dynamicTitle) {
      const routeTitle = router.currentRoute.value.meta?.title;
      const pageTitle =
        (routeTitle ? `${$t(routeTitle)} - ` : '') + preferences.app.name;
      useTitle(pageTitle);
    }
  });

  app.mount('#app');
}

export { bootstrap };
```

### 根组件 (app.vue)
```vue
<script lang="ts" setup>
import { computed } from 'vue';

import { useAntdDesignTokens } from '@vben/hooks';
import { preferences, usePreferences } from '@vben/preferences';

import { App, ConfigProvider, theme } from 'antdv-next';

import { antdLocale } from '#/locales';

defineOptions({ name: 'App' });

const { isDark } = usePreferences();
const { tokens } = useAntdDesignTokens();

const tokenTheme = computed(() => {
  const algorithm = isDark.value
    ? [theme.darkAlgorithm]
    : [theme.defaultAlgorithm];

  if (preferences.app.compact) {
    algorithm.push(theme.compactAlgorithm);
  }

  return { algorithm, token: tokens };
});
</script>

<template>
  <ConfigProvider :locale="antdLocale" :theme="tokenTheme">
    <App>
      <RouterView />
    </App>
  </ConfigProvider>
</template>
```
