# 后管前端项目总览规范

> **适用范围**：`frontend/vue-vben-admin/` 目录下的所有文件

## 一、强制要求（AI必须100%遵守，违反即错误）

1. **必须使用TypeScript**：所有文件必须使用TypeScript编写，禁止使用JavaScript
2. **必须使用Vue 3 Composition API**：所有组件必须使用`<script setup>`语法
3. **必须使用Ant Design Vue**：UI组件必须优先使用Ant Design Vue组件库
4. **必须遵循目录结构**：严格按照规定的目录结构组织代码
5. **必须使用requestClient**：所有API请求必须通过requestClient发起
6. **必须使用国际化**：所有用户可见文本必须使用`$t()`函数进行国际化处理
7. **必须使用Pinia**：状态管理必须使用Pinia，禁止使用Vuex或其他状态管理库
8. **必须使用VXE Table**：表格组件必须使用VXE Table，禁止使用其他表格组件

## 二、推荐做法（AI优先采用，效果更好）

1. **使用组合式函数**：将可复用逻辑提取到composables中
2. **使用TypeScript类型定义**：为所有API接口、组件Props、事件定义明确的类型
3. **使用命名空间**：API接口使用namespace组织相关类型和函数
4. **使用模块化路由**：路由配置按模块拆分，便于维护
5. **使用适配器模式**：使用adapter层封装第三方组件，统一接口
6. **使用错误边界**：关键页面添加错误处理和加载状态
7. **使用响应式设计**：页面布局必须支持响应式，适配不同屏幕尺寸

## 三、禁止行为（AI绝对不能生成）

1. **禁止硬编码文本**：所有用户可见文本禁止直接硬编码，必须使用国际化
2. **禁止直接使用axios**：禁止直接使用axios或fetch，必须使用requestClient
3. **禁止使用Options API**：禁止使用Vue 2的Options API语法
4. **禁止在组件中直接调用API**：API调用必须通过api层的封装函数
5. **禁止使用any类型**：禁止使用any类型，必须明确定义类型
6. **禁止在template中写复杂逻辑**：模板中的复杂逻辑必须提取到methods或computed中
7. **禁止使用全局样式污染**：组件样式必须使用scoped或CSS Modules

## 八、标准代码示例

### 应用入口文件
```typescript
// src/main.ts
import { createApp } from 'vue'
import { createPinia } from 'pinia'
import router from './router'
import { setupI18n } from './locales'
import App from './app.vue'

import '@vben/styles/index'

const app = createApp(App)
app.use(createPinia())
app.use(router)
await setupI18n(app)
app.mount('#app')
```
