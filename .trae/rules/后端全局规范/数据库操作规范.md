# 数据库操作规范

## 实体类设计

```java
@Data
@TableName("sys_user")
public class User extends BaseData {
    @TableId(value = "id", type = IdType.ASSIGN_ID)
    private Long id;

    @TableField("username")
    private String username;

    // 继承 BaseData: createTime, updateTime, deleted
}
```

## Mapper 接口

```java
public interface UserMapper extends BaseMapper<User> {

    // 复杂查询使用 @Select
    @Select("SELECT * FROM sys_user WHERE username = #{username}")
    User selectByUsername(@Param("username") String username);
}
```

## Service 层

```java
@Service
@RequiredArgsConstructor
public class UserServiceImpl implements UserService {

    private final UserMapper userMapper;

    public User getById(Long id) {
        return userMapper.selectById(id);
    }

    public PageResult<UserVO> pageUsers(PageParams params) {
        Page<User> page = this.page(
            new Page<>(params.getPageNo(), params.getPageSize())
        );
        return PageResult.of(page, this::convertToVO);
    }
}
```

## 分页查询

```java
// ✅ 分页查询（分页参数参考：PageParams.java）
public PageResult<UserVO> pageUsers(PageParams params) {
    Page<User> page = this.page(
        new Page<>(params.getPageNo(), params.getPageSize())
    );
    return PageResult.of(page, this::convertToVO);
}
```
