# 数据库操作规范

## 实体类设计

```java
@Data
@TableName("sys_user")
public class User extends BaseData  implements BasecopyProperties {
    @TableId(value = "id", type = IdType.ASSIGN_ID)
    private Long id;

    @TableField("username")
    private String username;

    // 继承 BaseData: createTime, updateTime, deleted
}
```

## Mapper 接口

```java
public interface UserMapper extends BaseMapper<User> {

    // ❌ 禁止：不允许使用 @Select 直接编写 SQL
    // @Select("SELECT * FROM sys_user WHERE username = #{username}")
    // User selectByUsername(@Param("username") String username);

    // ✅ 推荐：使用 MyBatis-Plus Lambda 查询
    // 复杂查询应在 Service 层通过 LambdaQueryWrapper 实现
}
```

## Service 层

```java
@Service
@RequiredArgsConstructor
public class UserServiceImpl implements UserService {

    private final UserMapper userMapper;

    public User getById(Long id) {
        return userMapper.selectById(id);
    }

    // ✅ 推荐：使用 LambdaQueryWrapper 进行条件查询
    public User getByUsername(String username) {
        return userMapper.selectOne(Wrappers.lambdaQuery(User.class)
            .eq(User::getUsername, username));
    }

    public PageResult<UserVO> pageUsers(PageParams params) {
        Page<User> page = this.page(
            new Page<>(params.getPageNo(), params.getPageSize())
        );
        return PageResult.of(page, this::convertToVO);
    }
}
```
