# 样式规范

## 基本规范

| 规范 | 说明 |
|------|------|
| 使用 scoped | 组件样式添加 `scoped` 避免污染全局样式 |
| 避免行内样式 | 优先使用 CSS 类 |
| 使用 CSS 变量 | 主题色等使用变量 |
| 移动优先 | 先写移动端样式 |
| 响应式设计 | 适配不同屏幕尺寸 |

## 命名规范

使用 BEM 命名规范或 kebab-case：

```scss
// BEM 规范
.user-card {
  &__header {
    font-size: 18px;
  }

  &__body {
    padding: 16px;
  }

  &--active {
    border-color: var(--primary-color);
  }
}

// kebab-case
.user-card { }
.user-card-header { }
.user-card-body { }
```

## Admin 项目样式

```scss
<style scoped lang="scss">
.user-card {
  padding: 16px;
  background: #fff;

  &__header {
    font-size: 18px;
    font-weight: bold;
  }

  &--active {
    border-color: var(--primary-color);
  }
}
</style>
```

## Blog 项目样式

```scss
// styles/variable.scss
$primary-color: #409eff;
$border-radius: 4px;

// styles/mixin.scss
@mixin flex-center {
  display: flex;
  justify-content: center;
  align-items: center;
}

// 组件样式
<style lang="scss" scoped>
.article-card {
  padding: 20px;
  border-radius: $border-radius;

  &__title {
    font-size: 16px;
    @include flex-center;
  }
}
</style>
```

## 响应式设计

```scss
// 移动优先
.container {
  width: 100%;
  padding: 16px;

  // 平板
  @media screen and (min-width: 768px) {
    width: 750px;
    padding: 24px;
  }

  // 桌面
  @media screen and (min-width: 1024px) {
    width: 960px;
    padding: 32px;
  }

  // 大屏
  @media screen and (min-width: 1280px) {
    width: 1200px;
  }
}
```

## CSS 变量使用

```scss
:root {
  --primary-color: #409eff;
  --success-color: #67c23a;
  --warning-color: #e6a23c;
  --danger-color: #f56c6c;
  --info-color: #909399;

  --border-radius: 4px;
  --border-color: #dcdfe6;
}

// 使用
.button {
  background-color: var(--primary-color);
  border-radius: var(--border-radius);
}
```

## 禁止行为

❌ **禁止**：在组件样式中使用 `!important`
❌ **禁止**：在模板中使用行内样式（除动态计算样式外）
❌ **禁止**：使用行内 `style=""` 绑定静态样式
